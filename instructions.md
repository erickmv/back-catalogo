# Instrucciones para ejecutar el proyecto

---

## Backend (Laravel)

### 1. Clone repository

git clone <https://github.com/erickmv/back-catalogo.git>

### 2. Install Depencencies

composer install

### 3. Run migrations

php artisan migrate

### 4. Run serve

php artisan serve


This will start the server at http://localhost:8000.
